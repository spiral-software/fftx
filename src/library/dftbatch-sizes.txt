##
##  Sizes of batch DFTs to build
##  Hash (#) as first non-white space indicates a comment
##  Lines containing white space only are ignored
##  All other lines must be valid size specs in the form:
##  nbatch := <bat>;  szns := <len>; stridetype := {1|2|3|4};
##  <bat> is a single value (representing the product of the grid size, e.g., 4x4 -> 16)
##  Stridetype determines the stride applied on reading & writing:
##      1 ==> write no stride, read no stride
##      2 ==> write no stride, read strided
##      3 ==> write strided, read no stride
##      4 ==> write strided, read strided
